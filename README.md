# Register panel BETA


### Comenzando beta 0.1 🚀
 
  > Este es un proyecto que aun se encuentra en BETA y seguimos desarollando poco a poco este panel 

## Pre-requisitos 📋
 - XAMPP
 - MySQL
 - Cookies activadas
 
## Instalación 🔧
  
  - Descarga el repositorio y archivos
  - Descomprimerlo en tu carpeta xampp/htdocs
  - abre tu localhost
  - Prueba :)
